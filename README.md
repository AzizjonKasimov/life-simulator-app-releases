# Life Simulator - Releases

Public download host for the **Life Simulator** Android app. The source code lives in
the public `AzizjonKasimov/life_simulator_app` repository; this repo holds release
APKs and the small version manifest read by the in-app updater.

- **`version.json`** - metadata for the latest version: version code/name, APK URL,
  and release notes. The app fetches this on launch to decide whether to offer an
  update.
- **APK downloads** - see the [Releases](../../releases) page.

To install manually, download the latest `LifeSimulator-x.y.z.apk` from Releases
and open it on your phone. Allow installs from this source when Android asks.
Once installed, the app can offer future updates itself.
